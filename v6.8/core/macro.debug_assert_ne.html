Macro core::debug_assert_ne

1.13.0 · source · 
macro_rules! debug_assert_ne {
    ($($arg:tt)*) => { ... };
}
Expand description

Asserts that two expressions are not equal to each other.

On panic, this macro will print the values of the expressions with their debug representations.

Unlike assert_ne!, debug_assert_ne! statements are only enabled in non optimized builds by default. An optimized build will not execute debug_assert_ne! statements unless -C debug-assertions is passed to the compiler. This makes debug_assert_ne! useful for checks that are too expensive to be present in a release build but may be helpful during development. The result of expanding debug_assert_ne! is always type checked.

Examples

let a = 3;
let b = 2;
debug_assert_ne!(a, b);
